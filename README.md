# Bird Card Blackjack

A mobile-friendly Blackjack web app featuring Tracey's hand-illustrated bird playing cards.

## Play

Visit [traceyami.github.io/bird-blackjack](https://traceyami.github.io/bird-blackjack)

## Rules

- Start with $500, fixed $25 bet per hand
- Blackjack pays 3:2
- Dealer hits on 16 or less, stands on 17+
- Double down doubles your bet for exactly one card

## Development

No build tools required. Open `index.html` in a browser.

## Fonts

Uses Copperplate via Adobe Fonts. Update the embed code in `index.html` with your own Web Project.
